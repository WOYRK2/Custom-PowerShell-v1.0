YOU MUST HAVE THE LATEST VERSION OF PYTHON INSTALLED
I would like to note upfront that this is still an early version, so there may be bugs, inaccuracies, and not all mechanics may be implemented yet.
How to use: open terminal as administrator, type cd (folder path), you need to copy the path of the folder containing all the necessary files for work, example - C:\Users\lesha\PycharmProjects\CustomTerminal. After that, type py main.py and follow the instructions! If you need to change the image, use any code editor to open main.py. Then find the variable OP (Note: does not support images!) and in TempData.Json change the variables Uses, UsesTwo, UsesThree to false, or delete the original JSON file and rename its copy, then run the setup again. You can use this code anywhere, but please mention the author!
